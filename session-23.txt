facts.yaml
==========

  hosts: local
  connection: local
  tasks:
  - name: print facts
    ansible.builtin.debug:
      msg: "{{ ansible_facts }}"

  - name: check redhat or not
    ansible.builtin.debug:
        msg: "Yes, package manager is RedHat"
    when: ansible_facts.os_family == "RedHat"

  - name: check Debian or not

search -- ansible command vs shell

to store the output of a command we can use register keyword

ansible-command-vs-shell.yaml
=============================
- name: ansible command vs shell module
  hosts: local
  connection: local
  tasks:
  - name: command module test
    #ansible.builtin.command: cat /etc/passwd | grep root
    ansible.builtin.shell: cat /etc/passwd | grep root
    register: users_output

  - name: print output
    ansible.builtin.debug:
      msg: "{{ users_output }}"

  - name: command module test
    #ansible.builtin.command: cat /etc/passwd | grep root
    ansible.builtin.shell: ls -ltr > files.txt
    register: files_output

//here we have redirected ls -ltr output to files.txt -- so output can't be printed on screen 

we can just see the exit code as success -- rc - 0
  
command to execute -- ansible-playbook -i inventory.ini ansible-command-vs-shell.yaml

in output -- stdout_line we can see output

for simple commands we can use command module
if command contains | , > then use shell module

in below program topics is -- string datatype
by converting string to list we can benefit
by using split filter we can convert string to list by delimiter ,
for convertion from list to string , list to map we can use filters
tagsMap is map/collection datatype, we want to convert it to list of map , for that use filter dict2items 
tagsList datatype is list we want to convert it to map
students.yaml file contains data in yaml , we want to convert it to json

filters.yaml
============

- name: demo on filters
  hosts: local
  connection: local
  vars:
    topics: linux, shell, ansible, terraform  // datatype--string 
    tagsMap:
      name: ansible
      project: roboshop
      component: catalogue
    tagsList: [
        {
            "key": "name",
            "value": "ansible"
        },
        {
            "key": "project",
            "value": "roboshop"
        },
        {
            "key": "component",
            "value": "catalogue"
        }
    ]
  tasks:
  - name: greeting
    ansible.builtin.debug:
      msg: "Hi Sivakumar, {{ GREETING | default('Good Morning') }}"

  - name: convert string to list
    ansible.builtin.debug:
      msg: "{{ topics | split(',') }}"

  - name: convert map to list
    ansible.builtin.debug:
      msg: "{{ tagsMap | dict2items }}"

  - name: convert list to map
    ansible.builtin.debug:
      msg: "{{ tagsList | items2dict }}"

  - name: read students yaml
    ansible.builtin.set_fact: # setting the variable dynamically
      student_yaml: "{{ lookup('file', 'students.yaml' ) | from_yaml }}"

  - name: convert YAML to JSON
    ansible.builtin.debug:
      msg: "{{ student_yaml | to_nice_json  }}"

 
command to execute - ansible-playbook -i inventory.ini filters.yaml
