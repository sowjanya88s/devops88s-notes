handlers -- is a directory
handlers/notifiers -- we can restart services by using handlers

handlers are notifiers, u can call task-b when task-a changes. 
for ex: when a configuration changes, u can restart the application by using handlers
by using notify keyword we can trigger handlers

if we made changes in service file then only we should restart service file & along with that we will do daemon-reload
don't do restarts without any purpose
because by restarting  application  it needs downtime (max -- 5)
playbook is idempotent -- running playbook for multiple times -- not good

nginx -- tasks -- main.yaml
===========================
- name: enable nginx 1.24
  ansible.builtin.command: dnf module enable nginx:1.24 -y

- name: install nginx
  ansible.builtin.dnf:
    name: nginx
    state: present

- name: Remove the entire directory
  ansible.builtin.file:
    path: /usr/share/nginx/html
    state: absent

- name: create html directory
  ansible.builtin.file:
    path: /usr/share/nginx/html
    state: directory

- name: download frontend
  ansible.builtin.get_url:
    url: https://roboshop-artifacts.s3.amazonaws.com/frontend-v3.zip 
    dest: /tmp/frontend.zip

- name: unarchieve frontend
  ansible.builtin.unarchive:
    src: /tmp/frontend.zip
    dest: /usr/share/nginx/html
    remote_src: yes

- name: remove default nginx
  ansible.builtin.file:
    state: absent
    path: /etc/nginx/nginx.conf

- name: copy nginx.conf
  throttle: 2
  ansible.builtin.template:
    src: nginx.conf.j2
    dest: /etc/nginx/nginx.conf
  notify: resatrt nginx

here whenever changes happened in nginx.conf handlers will restart nginx with help of notify keyword

deployment 
===========
1.remove old code & download the new code & unzip the new code 
2.copy new code
3.install dependancies
4. restart the server
the code related to above tasks is in app-setup.yaml

in import-role of app setup module use tags keyword and give tag name to it as deployment
for ex:
tags:
- deployment

when u pass tag=deployment tasks in app-setup module will be executed
