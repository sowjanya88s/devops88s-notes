mongodb -- should have connection from bastion on port 22, catalogue , user on port 27017

redis -- should have connection from bastion on port 22, cart, user on port number 6379

RabbitMQ -- should have connections from bastion on port 22, payment on port 5672

catalogue -- should have connections from bastion on port 22, backend-alb on port 8080

note: communication between components happens through backend-alb even if they have dependent on each other

user -- should have connections from bastion on port 22, backend-alb on port 8080

cart -- should have connections from bastion on port 22, backend-alb on port 8080

shipping -- should have connections from bastion on port 22, backend-alb on port 8080

payment -- should have connections from bastion on port 22, backend-alb on port 8080

backend_alb -- should accept connection from all the components i.e catalogue , user , cart , shipping , payment on port 80 & should accept connection from frontend_alb on port number 80

frontend_alb -- should accept connections from frontend on port number 80 & it should accept connections from internet on port 443

frontend -- should accepet connections from bastion on port number 22 & its hould accept connections from frontend_alb on port number 80

backend-alb is like mediator for all components

with the help of auto scaling & based on load the instances related to components may increase or decrease so the communication between components happens with the help of backend-alb
