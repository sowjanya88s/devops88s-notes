DNS Records
===============
TTL -> Time to Live -> How much time to live in cache or How much time to update
A -> Holds the IP address of your domain

if u keep on changing the ip address u have to keep TTL low -- 1sec
if u put it as 1sec -- disadvantage here is dns resolution takes time

in hostinger low TTL -- 60 sec

usually everyone keeps TTL as 3600S

todo major changes atleast 1 day before reduce TTL to 1sec

once everything is fine, take TTL back to 3600

goto AWS hosted zone -- create A record -- give ur server public ip here -- TTL -- 1 sec -- create

to check this use nslookup command -- nslookup domain-name -- u should see ur public ip here
